# The Draft Decider - Fantasy Football 2026

Your complete fantasy football draft companion. Compare draft scenarios side-by-side to make the tough decisions and maximize your projected points.

## Features

- Compare draft scenarios side-by-side
- Yahoo's 2026 Top 150 rankings
- Customize league scoring settings
- Track rounds and projected points
- Mobile-friendly with swipe gestures
